Recursive Intelligence — R³ Logo Kit (first pass)
===================================================

Accent color: #FFAA00 (Hugger orange)
Dark background: #000000
Light background: #FFFFFF

Files:
- R3_unboxed.svg         (header / large uses, dark background)
- R3_boxed.svg           (favicon / small uses, dark background)
- R3_unboxed_light.svg   (header / large uses on light background)
- R3_boxed_light.svg     (favicon / small uses on light background)

Notes:
- The superscript 3 is editable text in the SVG (Space Grotesk preferred; falls back to system sans).
  When preparing for print or if you want absolute consistency, convert it to outlines in your vector editor.
- For very small sizes (<= 24 px), use the boxed version.
- Clearspace: at least half the cap height of the "R" on all sides.
- Monochrome fallback: change the 3's fill to white/black to match context.
